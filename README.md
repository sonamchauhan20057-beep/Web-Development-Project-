# Bitcoin Real-Time Buy & Sell App 

This project is a **crypto web application** that displays **real-time Bitcoin (BTC) market data** and allows users to **simulate Buy and Sell actions** using live prices.

The app shows **real crypto market data**, but no real money or real trading is involved.

---

##  Problem Statement

- Beginners and students find it difficult to understand how **real-time cryptocurrency prices** work.
- Most people do not know how **Buy/Sell calculations** are done using live market data.
- Real trading platforms are complex and risky for learning purposes.
- There is a need for a **safe demo application** that shows real Bitcoin data without using real money.

---

##  Solution

This project solves the above problems by:

- Fetching **real-time Bitcoin price data** from public crypto APIs.
- Providing a **simple and user-friendly interface** to understand crypto prices.
- Allowing users to **simulate Buy and Sell actions** using live prices.
- Automatically calculating **profit and loss** based on market changes.
- Helping students learn **API integration, real-time data handling, and financial logic** safely.

---

## Features

-  Real-time Bitcoin price (USD / INR)
-  Live crypto market data using public APIs
-  Buy & Sell simulation
-  24-hour high / low and price changes
-  Automatic profit & loss calculation
-  Responsive design (mobile & desktop)

---

##  How It Works

- The application fetches **live Bitcoin market prices** from a public API (CoinGecko / Binance).
- Price data is refreshed automatically.
- Users enter an amount and choose **Buy or Sell**.
- All calculations are performed using **real-time prices**.

This is a **demo / educational project**, not a real trading platform.

---

##  Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **API:** CoinGecko / Binance Public API  
- **Backend (Optional):** Node.js, Express  
- **UI Framework:** Bootstrap / Tailwind CSS
